5-Question Pop Quiz (Based on Friday: Custom CSS, Zipping, Submission)

  1. In VS Code, what is the best way to prepare your final project folder for the HUB submission?

    **A:** Compress the folder into a **Zip** file.

  2. What is the specific attribute we use to link an external CSS file to our HTML page?

    **A:** `href` (within the `` tag).

  3. When overriding a Bootstrap class with custom CSS, what key principle must you ensure?

    **A:** Your custom CSS must have **higher specificity** or load **after** the Bootstrap CDN/file.

  4. Which Bootstrap utility class would you use to hide an element only on a small screen?

    **A:** `.d-sm-none`.

  5. What are the three main files/folders that absolutely must be included in your final Bootstrap project submission?

    **A:** HTML files, a CSS folder/file, and an Images folder (if used).
